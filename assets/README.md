# Assets Directory

This directory contains media assets for the bot.

## Binance QR Code

Place your Binance Pay QR code image here with the name:
- `binance_qr.jpeg`

You can generate a QR code for your Binance Pay ID at:
https://www.binance.com/en/qr

The QR code will be shown to users when they click "Show QR Code" in the support section.
